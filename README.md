- Q3. 학습노트는 제출 불가능이라고 돼있는데  과정 중간 회고로 공부한 것을 정리하는 건 안되나요?  
   Q4.  강의자료 포스팅 금지 & 필기자료 등 개인 공부 자료 활용 가능이면 강의 자료를 사진 찍어 올리는 것 말고 중요하다고 생각하는 것 타이핑, 타이핑 필기로 올리는 건 괜찮은 건가요?

  > A4. 네 직접적인 캡쳐 자료만 활용하지 않으시면 됩니다.
